An open source webform
backend for your JAMstack sites.

How it works

  1. Add 
    action="https://plentiform.io/form/{ id }
    to any HTML form on your website.
  2. Plentiform will autodiscover the structure of your fields, validate user input, and return any submission errors.
  3. Access your data through the app dashboard or consume it from the API we expose for your site.

Features

  • Open source
  • Written in Go
  • Can be self-hosted
  • Provides JSON API
  • Third party integrations
  • Validates submissions
  • Returns input errors
  • Verifies form structure
  • Works with any website
  • Easy to use web UI
